wunderpack & wunderpass

  • wunderpack jetpack - This allows wunderpacklets to dock to it.
  • wunderpass extension- This allows wundepacklets to execute code in the firefox extension scope. Use with caution

pack+pass wunderpacklets

  • wp-delicious-a - A one-click delicious bookmarker. Bookmarks to the "first" delicious user in the browser's saved passwords
  • wp-delicious-b - A one-click delicious bookmarker. Bookmarks to the "second" delicious user in the browser's saved passwords

pack-only wunderpacklets

  • wp-all - A wunderpacklet that shows up on all pages
  • wp-google - A wunderpacklet that shows up only on google webpages
  • wp-cnn - A wunderpacklet that shows up only on cnn webpages
*note: very little documentation currently - see source code or the post at jetpack google group
